🔢 Math Practice

💡 Good vs Bad Prompts for Math

✅ Good:

"Find the derivative of 3x⁴ - 2x³ step-by-step. Show all work and explain each step using the power rule."

❌ Bad:

"What's the derivative?"

✨ Key: Be specific, ask for step-by-step explanations, and mention the concept/rule to use.

🔬 Science Practice

💡 Good vs Bad Prompts for Science

✅ Good:

"Explain photosynthesis to a 10th grader. Include the chemical equation, the role of chlorophyll, and the two main stages."

❌ Bad:

"What is photosynthesis?"

Key: Be specific, ask for detailed explanations, request examples or real-world connections

💻 Computer Science

💡 Good vs Bad Prompts for CS

✅ Good:

"Explain how for loops work in Python with 2 simple examples. I'm a beginner."

❌ Bad:

"What are loops?"

✅ Good:

"I'm getting an 'IndexError' in this Python code. Can you identify the bug and explain how to fix it? [paste code]"

❌ Bad:

"My code doesn't work."

Key: Include programming language, paste code when debugging, specify skill level, ask for examples

📜 History

💡 Good vs Bad Prompts for History

✅ Good:

"Explain the main causes of World War I in simple terms. Include at least 3 causes and how they led to the war."

❌ Bad:

"Why did WWI happen?"

✅ Good:

"Compare and contrast the American Revolution and the French Revolution. Create a table showing their similarities and differences."

❌ Bad:

"Tell me about revolutions."

Key: Ask for causes and effects, request comparisons, analyze primary sources, create timelines with dates

💡 What Makes a Good Prompt?

  • ✓ Be specific about what you want
  • ✓ Include context and desired format
  • ✓ Use clear, complete sentences
  • ✓ Specify tone, length, or style if needed
  • ✗ Avoid vague requests
  • ✗ Don't use incomplete sentences
  • ✗ Avoid ambiguous language
