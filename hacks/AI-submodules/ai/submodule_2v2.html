Prompt Engineering

How AI Responds to Prompts

Generative AI like ChatGPT and Claude use two key technologies:

Natural Language Processing

Helps AI understand and respond to human language naturally, like talking to another person.

Machine Learning

Enables AI to learn from your input and improve its responses over time.

💡 Good vs Bad Prompts for Math

✅ Good:

"Find the derivative of 3x⁴ - 2x³ step-by-step. Show all work and explain each step using the power rule."

❌ Bad:

"What's the derivative?"

✨ Key: Be specific, ask for step-by-step explanations, and mention the concept/rule to use.

Computer Science

💡 Good vs Bad Prompts for CS

✅ Good:

"Explain how for loops work in Python with 2 simple examples. I'm a beginner."

❌ Bad:

"What are loops?"

✅ Good:

"I'm getting an 'IndexError' in this Python code. Can you identify the bug and explain how to fix it? [paste code]"

❌ Bad:

"My code doesn't work."

Key: Include programming language, paste code when debugging, specify skill level, ask for examples

History

                
            


            
            

                
💡 Good vs Bad Prompts for History

                

                    

                        

                            
✅ Good:

                            
Explain the main causes of World War I in simple terms. Include at least 3
                                causes and how they led to the war.

                        

                        

                            
❌ Bad:

                            
Why did WWI happen?

                        

                    

                    

                        

                            
✅ Good:

                            
Compare and contrast the American Revolution and the French Revolution. Create a
                                table showing their similarities and differences.

                        

                        

                            
❌ Bad:

                            
Tell me about revolutions.

                        

                    

                

                

                    
Key: Ask for causes and effects, request comparisons, analyze primary
                        sources, create timelines with dates

Prompt Writing Guidelines

                                
                                

                                    
Good vs Bad Example:

                                    

                                        

                                            
Good:

                                            
"Create a detailed learning plan for mastering Python. Include daily goals, recommended resources, practice projects, and how to track progress."

                                        

                                        

                                            
Bad:

                                            
"Help me learn Python."

                                        

                                    

                                


                                

                                    
What Makes a Good Prompt:

                                    
    
                                        
  • Be specific about what you want
    • 
                                        
  • Include context and desired format
    • 
                                        
  • Use clear, complete sentences
    • 
                                        
  • Specify tone, length, or style if needed
    • 
                                        
  • Avoid vague requests
    • 
                                        
  • Don't use incomplete sentences
    • 
                                        
  • Avoid ambiguous language
    • 
                                    

                                

                                
                                

                                    
Examples by Subject:

                                    

                                        

                                            
Math:

                                            
"Solve step-by-step: 2x² + 5x - 3 = 0. Explain each step."

                                        

                                        

                                            
Science:

                                            
"Explain photosynthesis to a 10th grader using simple language and an analogy."

                                        

                                        

                                            
Computer Science:

                                            
"Explain how for loops work in Python with 2 simple examples."

                                        

                                        

                                            
History:

                                            
"Explain the main causes of World War I in simple terms. Include 3+ causes."
